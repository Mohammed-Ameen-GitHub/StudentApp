Configuration

React_StudentApp

React_StudentApp folder consists of public and src folders.
src folder contains student folder, css and javascript files.
Student folder consists of all the component javascript files.

WebApi_StudentApp

WebApi_StudentApp folder consists of controllers, models, views and all initialization folders for webapi project.
